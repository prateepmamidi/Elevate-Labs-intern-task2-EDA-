# Titanic Dataset - Exploratory Data Analysis

## Objective
Perform Exploratory Data Analysis (EDA) on the Titanic dataset using statistics and visualizations.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Steps Performed
1. Loaded Dataset
2. Data Cleaning
3. Summary Statistics
4. Histograms
5. Boxplots
6. Correlation Matrix
7. Pairplot
8. Outlier Detection
9. Skewness Analysis
10. Insights Generation

## Key Findings
- Females survived more than males.
- First-class passengers had higher survival rates.
- Fare contains outliers.
- Fare and Passenger Class are correlated.

Titanic Dataset
